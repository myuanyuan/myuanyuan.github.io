vue router

首先我们通过 npm 安装

因为vue-router是vue的一个插件，我们需要用Vue.use(pluginContructor)注册到vue对象上。
(虽然vue-router内部会检测window.vue对象是否存在，如果存在则自动调用Vue.use()方法，需要手动调用来确保注册到Vue中。
在webpack等支持CommonJS规范的环境中，vue对象并不会暴露到全局的window对象中，而是通过module.exports形式输出，因此需要使用者手动注册)
